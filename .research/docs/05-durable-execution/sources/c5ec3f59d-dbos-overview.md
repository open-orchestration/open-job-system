# Welcome to DBOS!

DBOS is a library for building reliable programs.
Add a few annotations to your application to **durably execute** it and make it **resilient to any failure**.

### Features [​](https://docs.dbos.dev/\#features "Direct link to Features")

[**Lightweight Durable Execution** \\
\\
Annotate your code to make it resilient to any failure](https://docs.dbos.dev/why-dbos)[**Run Anywhere** \\
\\
Run DBOS workflows in any environment, or serverlessly deploy them to DBOS Cloud](https://docs.dbos.dev/production/checklist)[**Reliable Queues** \\
