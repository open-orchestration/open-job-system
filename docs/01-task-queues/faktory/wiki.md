### Faktory Enterprise

[Permalink: Faktory Enterprise](https://github.com/contribsys/faktory/wiki#faktory-enterprise)

Upgrade to Faktory Enterprise for more features and official support.

- [Purchase Faktory Enterprise](https://billing.contribsys.com/fent/new.cgi)
- [Ent Cron](https://github.com/contribsys/faktory/wiki/Ent-Cron) \- enqueue jobs periodically
- [Ent Metrics](https://github.com/contribsys/faktory/wiki/Ent-Metrics) \- send real-time usage metrics to Statsd for monitoring
- [Ent Redis Gateway](https://github.com/contribsys/faktory/wiki/Ent-Redis-Gateway) \- expose Faktory's Redis instance to the network
- [Ent Expiring Jobs](https://github.com/contribsys/faktory/wiki/Ent-Expiring-Jobs) \- expire jobs after a time period
- [Ent Unique Jobs](https://github.com/contribsys/faktory/wiki/Ent-Unique-Jobs) \- ensure jobs aren't duplicated for a time
- [Ent Installation](https://github.com/contribsys/faktory/wiki/Ent-Installation) \- how to install Faktory Enterprise
- [Ent Throttling](https://github.com/contribsys/faktory/wiki/Ent-Throttling) \- throttle queues to prevent worker overload or SLA violations
- [Ent Batches](https://github.com/contribsys/faktory/wiki/Ent-Batches) \- create complex asynchronous job workflows
- [Ent Tracking](https://github.com/contribsys/faktory/wiki/Ent-Tracking) \- track job execution progress
- [Ent Remote Redis](https://github.com/contribsys/faktory/wiki/Ent-Remote-Redis) \- use a remote Redis instance with Faktory
